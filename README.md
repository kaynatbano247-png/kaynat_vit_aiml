This is my AIML vityarthi project
My name is Kaynat Bano and i have created a Smart Study Planner which is a Python-based application designed to help students efficiently plan and
manage their exam preparation. It organizes subjects based on deadlines and difficulty levels to create a structured study plan.
The features are:
1-add multiple subjects
2-set deadlines for ech subjects
3-assign difficulty levels
4-generates hours of study per subject
Technologies used:
1-Python
2-basic data structures(lists,loops)
Working:
user enter the number of subjects, subjects name, deadline, difficulty levels and no. of study hours available per day then program processes the
data and generates a study plan based on which has highest to lowest priority.
Learning outcomes:
1-understanding of problem-solving using python
2-basic project structure
3-introduction to a real-world application development.
